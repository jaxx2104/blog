# Project Brief

## Overview
Gatsby を使用した個人ブログサイト。技術記事や日記などを投稿することを目的としている。

## Key Features
- Markdown ベースの記事管理
- レスポンシブデザイン
- ダークモード対応
- カテゴリ/タグ機能
- プロフィールページ
- SEO 最適化

## Technical Stack
- Gatsby (React ベースの静的サイトジェネレータ)
- TypeScript
- styled-components
- GraphQL
- Markdown

## Content Structure
- content/posts/ 配下に年-月-日-タイトル形式で記事を管理
- 各記事は Markdown ファイルと画像で構成
- 記事のメタデータは Markdown の frontmatter で管理

## Development Setup
- Node.js 環境
- pnpm パッケージマネージャ
- ESLint/Prettier によるコードフォーマット
